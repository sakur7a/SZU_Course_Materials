# 常见QA
1. 转专业考试难度？要复习到什么程度？

    我实在比较难回答，客观上的确不难，和学校的课差不多。至于复习到什么程度...看个人吧，其实应该转专业的大多数人心里都是没底的，毕竟转失败了很可能就要在原专业继续读，大二转的话过程中压力也蛮大。我复习到后面倒是还好了，因为我做了太多题。所以总结就是，如果你会觉得没有把握，那就继续刷题！建议大家都有一定会成功的心态，我当时是觉得如果只有一个名额也应该是我（~~虽然考完之后马上觉得g了~~）。

2. 面试

    不要有过大的压力，如果进面的话其实已经差不多成功了。我面试的时候因为蛮紧张，问题几乎一个都没答出来。如果不会的话直接说不会就行，毕竟你应该才大一，懂得不多也是正常的。
